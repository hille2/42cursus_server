Если ты, дорогой пир, сел за новый мак, на котором нет твоего докера, не печалься.
Теперь у тебя есть супер(нет) скрипт, которой поможет тебе проделать всю монотонную и скучную работу гораздо быстрее.
Следуя простой инструкция ты сократишь себе время, удачи:

1. Clear memory
	~bash clear.sh
2. Delete "docker.io" and "Virtualbox"
	~rm -rf ~/.docker

3. Create a personal dir and symbolic links in goinfre
	~bash docker.sh
4. install docker(Manager Software Center)
   install virtual box(Manager Software Center)
5. ~bash new_mac.sh
